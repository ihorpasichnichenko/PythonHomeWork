

Indentation Error -  Возникает при неправильном отступе. Для воспроизведения необходимо не соблюдать отступы при написании например, условий.(IF,ELIF,ELSE..)

Name Error - Возникает, когда переменная не найдена. Для воспроизведения необходимо не обозначить переменную,или не дать ей значение, и в дальнейшем попытаться её использовать.

Syntax Error - Возникает при не верном написании синтаксиса. Для воспроизведния необходимо не закрыть скобку в строке со значениями,не указать двоеточие перед условиями,не взять стринг в кавычки...

Type Error - Возникает при использовании не верного типа данных. Для воспроизведения например, необходимо воспроизвести любую математическую операцию числа и строки без преобразования. 

Value Error - Возникает при передаче в функцию аргумента с неподдерживаемым значением. Для воспроизведения необходимо например, в функцию int(input('....:')) ввести не целое число,или буквенные значения.
